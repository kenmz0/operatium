# Backend API – Proyecto de Aprendizaje
Proyecto en desarrollo (learning project)
Este proyecto es un API backend desarrollado con NODE.js, express y ProstgreSQL como parte de un proceso de aprendizaje.

# Tecnologías
- Node.js
- Express
- PostgreSQL
- dotenv
- JavaScript

# Objetivos de aprendizaje
1. Levantar un servidor backend con Node.js y el framework express
2. Manejar variables de entorno .env
3. Conectar una api a un servicio de base de datos, en este caso PostgreSQL

## Estado actual
- Servidor Express operativo (Endpoint `/health`) 
- Variables de entorno configuradas
- Conexión a PostgreSQL (pendiente de uso en endpoints)
- Modelo de usuario definido
 

### Proximo pasos
1. Implementar registro y login de usuarios.
2. Aplicar hash de contraseñas (bcrypt).
3. Implementar autenticación basada en JWT.
4. Crear middleware de autenticación para proteger rutas.

## Roadmap
- [x] Base backend
- [ ] Usuarios y autenticación
- [ ] Autorización por roles
- [ ] Endpoints protegidos
- [ ] Despliegue


# Ejecución
1. Clonar el repositorio
2. Instalar dependencias:
   npm install
3. Crear archivo .env
4. Eejcutar
    npm run dev
